installation
""""""""
``git clone https://github.com/9kin/cfdl``

``cd cfdl``

``git submodule init``

``git submodule update``

``python3 -m venv env # if you want use venv``

``source env/bin/activate # if you want use venv``

``pip3 install .``


download task (tutorials) with asyncio
